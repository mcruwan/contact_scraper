OPTION 1: Environment Variable (Recommended)
---------------------------------------------
1. Open your terminal
2. Run this command (replace with your actual key):

   export OPENROUTER_API_KEY='sk-or-v1-your-key-here'

3. To make it permanent, add this line to your ~/.zshrc or ~/.bash_profile:

   echo "export OPENROUTER_API_KEY='sk-or-v1-your-key-here'" >> ~/.zshrc
   source ~/.zshrc


OPTION 2: Create a .env file
-----------------------------
1. In this directory (/Applications/MAMP/htdocs/contact_scraper/)
2. Create a file named: .env
3. Add this line:

   OPENROUTER_API_KEY=sk-or-v1-your-key-here


GETTING YOUR API KEY
--------------------
1. Visit: https://openrouter.ai
2. Sign up (free)
3. Go to: https://openrouter.ai/keys
4. Click "Create Key"
5. Copy the key (starts with sk-or-v1-...)
